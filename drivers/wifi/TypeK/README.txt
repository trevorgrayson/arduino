TypeK library
-------------

Copyright (C) 2010 MLG Properties, LLC
Released under BSD license (see files).

Provides support for ITS-90 thermocouple linearization,
Type K thermocouples.

Create a folder and place these files in
C:/Program Files/arduino-0018/libraries/TypeK on your computer.

Includes functions to:
 - convert a mV reading into an absolute temperature (C or F)
 - convert a mV reading + ambient temp reading to a compensated sensor temperature
 - convert a temperature (C or F) into the corresponding mV value

See TypeK.h for details.
